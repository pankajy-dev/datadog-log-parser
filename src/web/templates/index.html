CSV File Format

                
Upload a CSV file with a content column containing logs. Common formats from Datadog, CloudWatch, etc. are supported.

Redaction

                
When enabled, sensitive fields (keys, tokens, passwords) are redacted, keeping only the first and last N characters.

                
Example: "api_key": "abcd123...xyz"
